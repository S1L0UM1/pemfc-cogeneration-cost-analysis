# PEMFC Cogeneration Cost Analysis

This repository contains Python scripts developed to analyze the economic
behavior of a PEM fuel cell cogeneration system for an isolated chalet.

## Objectives
The objective of this work is to study how the cost of the PEMFC cogeneration
technology evolves with:
- Battery storage capacity
- Hydrogen storage tank size

## Scripts
- npc_vs_battery.py : NPC evolution versus battery capacity
- npc_vs_h2_tank.py : NPC evolution versus hydrogen tank size

## Requirements
Install required Python packages using:
```bash
pip install -r requirements.txt
python npc_vs_battery.py
python npc_vs_h2_tank.py
